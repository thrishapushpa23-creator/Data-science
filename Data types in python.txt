Data types in python


Single variables


1.int – Integer (e.g., 10)

2.float – Decimal number (e.g., 3.5)

3.complex – Complex number (e.g., 2+3j)

4.str – String/text (e.g., "Hello")


Multi variables

1.list – Ordered, changeable (e.g., [1, 2, 3])

2.tuple – Ordered, not changeable (e.g., (1, 2, 3))

3.range – Sequence of numbers (e.g., range(5))

4.set – Unordered, no duplicates (e.g., {1, 2, 3})

5.dict – Key-value pairs (e.g., {"a": 1})




Operators in python

1.Arithmetic Operators

Addition (+) 

Subtraction (-) – Example: 5 - 3 → 2

Multiplication (*) –Example: 5 * 3 → 15

Division (/) –  Example: 5 / 2 → 2.5

Floor Division (//) –  Example: 5 // 2 → 2

Modulus (%) – Example: 5 % 2 → 1

Exponentiation (**) – Example: 5 ** 3 → 125

2. Comparison Operators
Used to compare values (result is True/False)

== (Equal)

!= (Not equal)

> (Greater than)

< (Less than)

>= (Greater or equal)

<= (Less or equal)

3. Logical Operators
Used with conditions

and → True if both are true

or → True if at least one is true

not → Reverses the result

4. Assignment Operators
Used to assign values
=

+=      Addition Assignment Operator

-=      Subtraction Assignment Operator

**=     Exponentiation Assignment Operator

/=      division Assignment Operator

%=      Modulus Assignment Operator

//      Floor Division Assignment Operator

*=      Multiplication Assignment Operator

5. Bitwise Operators
Work on binary values

& (AND)
| (OR)
^ (XOR)
~ (NOT)
<<(LEFT SHIFT)
>>(RIGHT SHIFT)

6. Membership Operators
Check if value exists in a sequence

in(TRUE IF FOUND)

not in(TRUR IF NOT FOUND)

7. Identity Operators
Check if two variables refer to same object

is(TRUE If IDENTICAL)

is not(TRUE IF NOT IDENTICAL)
